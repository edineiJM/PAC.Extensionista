Introdução à Linguagem C

Vamos explorar a introdução à linguagem C e o passo a passo para aprender essa linguagem fundamental. É uma das mais influentes e amplamente utilizada na programação, especialmente para desenvolvimento de sistemas e aplicativos de baixo nível.

A linguagem C foi criada por Dennis Ritchie na década de 1970 nos Laboratórios Bell. É conhecida por sua eficiência e controle de baixo nível sobre hardware, o que a torna ideal para sistemas operacionais, compiladores e aplicações que requerem desempenho.

Biblioteca Basica

Em C, as bibliotecas são coleções de funções e macros que podem ser usadas para realizar várias operações, como manipulação de strings, operações matemáticas, controle de entrada/saída, manipulação de arquivos, entre outras. Elas ajudam a evitar que os programadores tenham que reinventar a roda ao criar código para funcionalidades comuns.
Aqui estão algumas das bibliotecas básicas mais utilizadas na linguagem C:

1. < stdio.h > (Standard Input/Output Library)
A biblioteca padrão para entrada e saída. Ela fornece funções para leitura e escrita de dados.

Funções principais:

printf: Escreve dados na saída padrão (geralmente a tela).

scanf: Lê dados da entrada padrão (geralmente o teclado).

fprintf: Escreve dados em um arquivo ou stream específico.

fscanf: Lê dados de um arquivo ou stream específico.

fopen, fclose: Abre e fecha arquivos.

fread, fwrite: Lê e escreve dados binários em arquivos.

fgets, fputs: Lê e escreve strings em arquivos.

Coleta de Informações

Para coletar informações do usuário em um programa e usá-las depois, precisamos pedir essas informações de uma forma específica. Para isso, usamos algo chamado scanf, que segue o padrão scanf("x", &y);. Aqui, x representa o tipo de dado que queremos coletar, como texto ou números, e y é o nome da caixinha (variável) onde vamos guardar essa informação.

Tipos de Dados:

int: Números inteiros (ex: 5, -3).
float: Números com ponto decimal (ex: 3.14).
char: Um único caractere (ex: 'a').
double: Números com mais precisão

Declaração de variável

Declarar variáveis em programação é como dar nome a caixinhas onde guardamos diferentes tipos de informações. Cada caixinha tem um nome e um tipo específico de coisa que pode guardar, como números inteiros, números com vírgula ou palavras.
Por exemplo, se você quiser guardar a idade de alguém, que é um número inteiro, você cria uma caixinha do tipo int e dá a ela um nome, como idade. Assim, você diz ao computador que aquela caixinha vai guardar um número inteiro.
Da mesma forma, se quiser guardar uma altura, que pode ter vírgula (como 1.75), você usa uma caixinha do tipo float. E, se precisar guardar o nome de uma pessoa, você usa uma caixinha chamada char, que é usada para armazenar letras ou uma sequência de letras.
Depois de criar as caixinhas, você pode colocar valores nelas, se você declarou uma variável chamada idade, pode guardar o valor 10 nela. Da mesma forma, se você declarou uma variável chamada altura, pode guardar o valor 1.75.
No final das contas, declarar variáveis é basicamente dizer ao computador: "Ei, eu vou precisar guardar essa informação aqui, e é desse tipo, então crie um espaço para ela!". Isso ajuda o computador a organizar e usar as informações de maneira eficiente durante o programa.

Condicional

Agora vamos para condicional, que ira verificar se a informação obedece uma instrução, efetue uma sequencia de códigos, caso contrario, faca outra, veja no exemplo a seguir:

Vamos utilizar sempre if (verifica se) ou else (para caso a informação se não seja atendida). Vamos solicitar a idade de uma pessoa, se ela for maior de 18 anos, o programa deve imprimir MAIOR DE IDADE, caso contrario, MENOR DE IDADE.

int main(){
int idade;
printf(“Digite sua idade”);
scanf(“%i”, &idade);

if (idade>=18)
printf(“MAIOR DE IDADE”);
else
printf(“MENOR DE IDADE”);
return 0;
}

Então nesse caso, ele ira solicitar a idade, e verificar se a idade informada é maior ou igual a 18, e ira imprimir MAIOR DE IDADE, caso contrario MENOR DE IDADE.

Laços de repetição

Os laços de repetição são ferramentas fundamentais na programação que permitem que um computador execute um bloco de código várias vezes sem precisar reescrever o mesmo código repetidamente. Eles são muito úteis quando precisamos repetir uma ação, seja um número fixo de vezes ou até que uma condição específica seja atendida.

Existem três tipos principais de laços em C:

Laço for: Este laço é ideal quando sabemos exatamente quantas vezes queremos repetir uma ação. Ele possui três partes principais: o ponto de partida (onde a repetição começa), a condição (que determina quando o laço deve parar) e a atualização (que ajusta a variável de controle a cada repetição). O laço for é eficiente e fácil de entender, pois todos esses elementos estão reunidos em uma única linha.

Laço while: Este tipo de laço é usado quando não sabemos exatamente quantas vezes vamos repetir a ação, mas sabemos que queremos continuar enquanto uma condição for verdadeira. O laço while verifica essa condição antes de executar o código dentro dele. Isso significa que, se a condição for falsa desde o início, o código dentro do laço não será executado nem uma vez.

Laço do...while: O laço do...while é semelhante ao while, mas com uma diferença importante: ele garante que o bloco de código seja executado pelo menos uma vez. Isso ocorre porque a condição é verificada após a execução do código. Portanto, mesmo que a condição seja falsa logo no início, o código dentro do laço será executado uma vez antes que a verificação aconteça.
